ngSwitch

  1. ngSwitch is actually comprised of two directives, and atttribute directive and a structural directive. It's very similar to a switch statement in Javascript and other programming languages, but in the template.
  2. The ng-switch directive lets you hide/show HTML elements depending on an expression.
  3. We can also define a default section, by using the ng-switch-default directive, to show a section if none of the other sections get a match.
  4. Three things to keep in mind - ngSwitch, ngSwitch, ngSwitchDefault

<select (change)="fun($event)">
<option value="fridge">fridge</option>
>option value="TV">TV</option>
</select>

<div [ngSwitch]="choose">
<h3 *ngSwitchDefault>Please Choose</h3>
<h3 *ngSwitchCase="'fridge'">fridge</h3>
<h3 *ngSwitchCase="'TV'">TV</h3>
</div>

choose="";
fun(val:any){
this.choose=val.target.value;
}
