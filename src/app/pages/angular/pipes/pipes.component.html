Pipes

  1. Angular Pipes can be used to transform data to desired output.
  2. Pipes takes in a data input and transforms data to a different output.
  3. Using the Pipe operator (|), we can apply the Pipe's features to any of the property in our Angular Project.
  4. Pipes (|) in Angular are used to transform the data before displaying it in a browser. Angular provides a lot of built-in pipes to translate the data before displaying it into the browser and as we know, Angular lets us extend its feature, we can even create custom pipes in Angular.
  5. For example, pipe can take "Learn Angular" as input and transform to "LEARN ANGULAR" in capital or you can give input as "100" which you want to display (transform) as "100.00 $" on UI.
  6. syntaxes are written inside HTML
    Input Data------------------------->Filter--------------------->Final output { {'Test' |uppercase} }--------------->TEST
  7. Types of Pipes
    1. Built-In Pipes
    2. Parameterized Pipes
    3. Chaining Pipes
    4. Custom Pipes

String Pipe

  1. Uppercase: Converts the string type data to upper case in the UI.
  2. Lowercase: Converts the string type data to lower case in the UI.
  3. Titlecase: Converts the string type data in which the first alphabet of each word is made capital and the rest will be in the small case.
  4. Slice: <starting index> It works on string fields or Array. It slices the input from the index till the end of the string or array length.
  5. Slice: <starting index>:<end index> It workson string fields or array, It slices the input from the starting index to the end index. (excluding the end index)

Parameterized Pipe

  1. We can pass any number of paramters to the pipe using a colon (:)
  2. data | pipeName: parameter 1 : parameter 2 : parameter 3 ... : parameter n
  3. Syntax

    Date- Short:-{ {DOB | date:"short"}}

  4. Currency - { {salary | currency:'USD':true}}

in .ts file
dob=new Date(1986,8,2);
salary:number=65000;

in .html file
Medium: { {dob|date:"medium"}}
Specific Format : { {dob|date:"dd/MM/yyyy"}}
Medium Time: { {dob|date:"mediumTime"}}
Salary :- { {salary|currency:"USD":true}}

Chaining Pipe

  1. We can use multiple pipes with the same data at the same. This also referred to as chaining pipes.
  2. data | pipe 1 | pipe 2 | pipe 3 | ....... pipe n
  3. data | pipe 1: parameter 1: parameter 2: parameter 3 ....: parameter n | pipe 2 : parameter 1: parameter 2: parameter 3....: parameter n| pipe 3: parameter 1: parameter 2: parameter 3.... : parameter n | ....... pipe n: parameter 1: parameter 2: parameter 3.... : parameter n
  4. .ts file
    dob=new Date(1986,8,2);

    .html file <span> { { dob|date:"medium" | lowercase}}</span>

Custom Pipe

  1. To create a pipe in Angular 7, you have to apply the @Pipe decorator to class, which we can import from the core Angular library.
  2. The @Pipe decorator allows you to define the pipe name that you'll use within template expressions.
