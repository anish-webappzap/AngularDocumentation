Property Binding

  1. It is used to bind values of component/model properties to the HTML element.
  2. Depending on the values, it will change the existing behavior of the element.
  3. Syntax - [property]='expression'
  4. In property binding, there is source and target. For this example, we can define it as [innerHTML]='firstname'. Here, innerHTML is a target is a property of span tag and source is a component property i.e. firstname
in .ts file
title="Hello World";

in .html file
<span [innerHTML]="title"><span>
