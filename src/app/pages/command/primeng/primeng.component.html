For install Primeng
Enter these Command: 

npm install primeng --save
npm install primeicons --save

Now open angular.json file and paste these style's links: 

. . . .
 "styles": [ 
   "./node_modules/primeicons/primeicons.css",
   "./node_modules/primeng/resources/themes/lara-light-blue/theme.css",
   "./node_modules/primeng/resources/primeng.min.css",
   "src/styles.scss"
],
. . . .

Now Primeng is installed on your project.

You can go ahead and use component

For that import module which you want use import that in module.ts file
