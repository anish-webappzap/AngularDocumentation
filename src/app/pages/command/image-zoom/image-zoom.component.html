Image Zoom

Here we are goig use to of Image Zoom on Hover
for that we are using image-zoom library

Note: Angular Version must be 14 
npm i --save ngx-image-magnifier

Import this module 

import { NgxImageZoomModule } from 'ngx-image-zoom';
...
imports: [
     NgxImageZoomModule
 ],
...

Open component where you want use:

<lib-ngx-image-zoom 
thumbImage="thumbImagePath"
fullImage="fullImagePath"></lib-ngx-image-zoom>
