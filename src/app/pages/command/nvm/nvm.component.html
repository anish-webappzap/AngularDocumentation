NVM (Node Version Manager) is allows you to install different versions of Node,
and switch between these versions depending on the project that you're working
on via the command line.

Install Node from link : Download

After complete installtion open CMD and we assume that you want change version from 16
to 14 then enter this below commmands : 

nvm ls
nvm install 14
nvm use 14

Now you successfully migrate on Node 14 version
